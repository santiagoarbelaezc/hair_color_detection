# hair_color_detection

Proyecto de visión por computadora para la detección y modificación del color de cabello usando YOLOv8 con segmentación.

## Estructura
* `notebook/`: Contiene los Jupyter Notebooks para ejecución local y en Google Colab.
* `data/`: Directorio ignorado por git donde deben ir las imágenes (CelebA-HQ).
* `results/`: Resultados de inferencia y pesos del modelo.
